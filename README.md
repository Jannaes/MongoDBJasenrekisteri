# Jäsenrekisteri

Yksinkertainen jäsenrekisteri Node.js:llä, Expressillä ja MongoDB:llä.

## Ominaisuudet

- Lisää jäsen
- Hae jäsenet
- Muokkaa jäsentä
- Poista jäsen
- Swagger API -dokumentaatio

## Käynnistys

Asenna paketit:

npm install

Käynnistä palvelin:

node server.js

## Swagger

http://localhost:3000/api-docs


## Tietokanta

MongoDB:n export löytyy tiedostosta:
jasenrekisteri.jasenet.json

Import MongoDB Compassilla.
